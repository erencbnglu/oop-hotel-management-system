# Otel Yönetim Sistemi (C++)

Bu proje, C++ dili kullanılarak geliştirilmiş **ilk nesne yönelimli (OOP) projelerden biri** olarak hazırlanmıştır.  
Amaç; sınıf (class), nesne (object), kapsülleme (encapsulation) ve dosya işlemleri gibi temel C++ kavramlarını
uygulamalı olarak öğrenmek ve pekiştirmektir.

Program basit bir **otel yönetim sistemi** mantığıyla çalışır.  
Odalar, müşteriler ve işlemler sınıflar üzerinden yönetilir; bilgiler `.dat` dosyaları kullanılarak saklanır.

## Projede Kullanılan Temel Kavramlar
- Nesne Yönelimli Programlama (Class / Object)
- Constructor & Destructor
- Dosya okuma / yazma (`fstream`)
- Menü tabanlı konsol uygulaması
- Modüler kod yapısı (header + source dosyaları)
